# Quick Start Guide

## Try the Demo (2 minutes)

### Option 1: Full Working Demo ⭐ RECOMMENDED
1. Visit: [Kaggle Notebook](https://www.kaggle.com/code/donlapidos/medgemma-clinical-intelligence-aki-detection)
2. Click **"Copy & Edit"**
3. Enable GPU: Settings → Accelerator → GPU P100
4. Click **"Run All"**
5. Wait 3-5 minutes for model to load
6. Scroll down to see MedGemma analyzing real ICU patients!

### Option 2: UI Showcase
1. Visit: [Streamlit App](https://medconnect-intelligence.streamlit.app)
2. Explore the interface
3. View MIMIC-III validation results
4. See sample clinical cases

## What You'll See

- **AKI Detection:** Drug-induced kidney injury analysis
- **Sepsis Recognition:** SIRS criteria calculation & severity assessment
- **Clinical Reasoning:** Explainable AI (not black box)
- **Real Data:** MIMIC-III ICU patients validated
